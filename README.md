# Compression LZW

Développement d'un algorithme de compression et de décompression
**Lempel-Ziv-Welch** dans le cadre d'un Projet d'Algorithmique.

## Pour commencer

Texte lambda pour combler le vide avant l'intégration des premières commandes.

 ```
 lzw c entrée sortie
 ```
